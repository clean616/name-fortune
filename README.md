# 이름 운세 🔮

한국어 이름을 입력하면 당신의 사후 행선지를 알려주는 재미용 웹앱입니다.

## 기능

- 한글 이름 입력 후 자모 분해 애니메이션 재생
- 이름에 따라 4가지 행선지 중 하나 결정 (꽃나라 🌸 / 별나라 ⭐ / 천국 ☁️ / 지옥 🔥)
- 결과 URL 공유 기능 (`?name=홍길동` 파라미터로 직접 접근 가능)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```

## 기술 스택

- React 18
- Vite
- CSS Modules
